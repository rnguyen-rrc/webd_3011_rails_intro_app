5.3	Bootstrap Grid
	A Bootstrap container is used to center and constrain the page content
	Inside the container, a row is used to organize content horizontally
	Each recipe is placed inside a column using responsive grid classes
		<div class="col-sm-6 col-md-4 col-lg-3">
	These classes define how many columns appear on different screen sizes:

		col-sm-6 → 2 columns on small screens
		col-md-4 → 3 columns on medium screens
		col-lg-3 → 4 columns on large screens

	This allows the layout to automatically adjust based on screen size, ensuring a responsive design.
